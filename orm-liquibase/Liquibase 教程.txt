Liquibase:
 - 开源数据库架构变更管理解决方案, 实现数据库的迁移和适配schema变更
   An open-source database schema change management solution. Enables to manage revisions of your database changes easily.
 - Liquibase 4.5.0版本存在安全问题，最低升级到4.13.0版本
 - Oracle is better covered by liquibase, so certain features might not work on Sybase
